# 대한민국 특허출원서

## 발명의 명칭
노이즈 조건부 이중 경로 컨볼루션 및 양방향 시간축 상태 공간 모델을 이용한 영상 인식 장치 및 방법

## 영문 명칭
Apparatus and Method for Image Recognition Using Noise-Conditioned Dual-Path Convolution and Bidirectional Temporal State Space Model

---

## 기술분야

본 발명은 영상 인식 기술에 관한 것으로, 더욱 상세하게는 조도 변화, 안개, 노이즈 등 열악한 환경에서도 강건한 영상 인식을 수행하기 위한 노이즈 조건부 이중 경로 컨볼루션(NC-Conv) 블록 및 양방향 시간축 노이즈 조건부 상태 공간 모델(Bidirectional NC-SSM)을 이용한 영상 인식 장치 및 방법에 관한 것이다.

---

## 배경기술

### 종래 기술의 문제점

자율주행, ADAS(첨단 운전자 보조 시스템), 영상 감시 등의 분야에서 영상 인식 모델은 터널 진입/탈출, 야간, 안개 등 열악한 조건에서도 안정적으로 동작해야 한다.

종래의 경량 CNN 모델(MobileNet, EfficientNet 등)은 입력 품질에 관계없이 동일한 연산을 수행하므로, 열화된 입력에 대해 성능이 급격히 저하되는 문제가 있다. 특히, SE-Net, 동적 컨볼루션(Dynamic Convolution) 등 입력 의존적 채널 게이팅을 사용하는 모델은 깨끗한 입력에서는 높은 표현력을 보이나, 노이즈가 있는 입력에서는 곱셈적 노이즈 증폭(multiplicative noise amplification)으로 인해 O(sigma_n^4) 이상의 출력 분산을 야기한다.

또한, 종래의 영상 강건성 기법(AugMix, DeepAugment 등)은 학습 데이터를 다양화하는 데이터 증강(data augmentation) 방식으로, 모델 아키텍처 자체의 노이즈 강건성을 개선하지는 못한다.

동영상 처리에 있어서는, 기존 프레임 독립적(frame-independent) 방식은 각 프레임을 개별 처리하므로, 터널 내부와 같이 심하게 열화된 프레임에서 인접 깨끗한 프레임의 정보를 활용할 수 없는 한계가 있다.

### 선행기술

1. 오디오 NC-SSM (본 발명자의 선행 연구): 음향 신호의 서브밴드별 SNR에 기반하여 선택적(Selective) SSM과 고정(LTI) SSM 동역학을 보간하는 기술. 노이즈 분산을 O(sigma_n^6)에서 O(sigma_n^2)로 저감.
2. SE-Net (Hu et al., CVPR 2018): 채널 어텐션 기반 입력 의존적 가중치 재보정.
3. Dynamic Convolution (Chen et al., CVPR 2020): 입력 의존적 커널 어텐션.
4. AugMix (Hendrycks et al., ICLR 2020): 다양한 증강 혼합을 통한 강건성 향상.

## 발명의 내용

### 해결하고자 하는 과제

본 발명은 다음의 과제를 해결하고자 한다:

1. 입력 품질에 따라 컨볼루션 연산의 동작을 적응적으로 변경하여, 깨끗한 입력에서는 높은 표현력을, 열화된 입력에서는 높은 강건성을 동시에 달성하는 영상 인식 장치 및 방법을 제공한다.
2. 동영상에서 시간축 양방향 문맥 전파를 통해 열화된 프레임의 인식 정확도를 회복하는 방법을 제공한다.
3. MCU(마이크로컨트롤러) 급 초경량 하드웨어에서 실시간 동작 가능한 영상 인식 모델을 제공한다.
4. 별도의 품질 라벨(supervision) 없이 입력 품질을 자동으로 추정하여 연산을 조건부 제어하는 방법을 제공한다.

### 과제의 해결 수단

#### 제1 발명: NC-Conv 블록 (노이즈 조건부 이중 경로 컨볼루션)

본 발명의 NC-Conv 블록은 다음 구성요소를 포함한다:

**(a) 정적 경로 (Static Path)**
- 고정 가중치 깊이별 분리 컨볼루션(Depthwise Separable Convolution) 및 배치 정규화(Batch Normalization)로 구성
- 입력에 무관한 고정 연산으로, 노이즈 분산 O(sigma_n^2) 유지
- 열화된 입력에서 강건한 특징 추출 담당

**(b) 동적 경로 (Dynamic Path)**
- 깊이별 분리 컨볼루션, 배치 정규화, 및 입력 의존적 채널 게이트(channel gate)로 구성
- 채널 게이트 alpha(x) = Sigmoid(FC(SiLU(FC(GAP(x)))))
- 입력 의존적 연산으로 높은 표현력을 제공하나, 노이즈 분산 O(sigma_n^{4+})

**(c) 시그마 게이트 (sigma gate)**
- 입력 특징맵으로부터 품질 점수 sigma in [0,1]을 산출하는 경량 네트워크
- sigma = Sigmoid(FC(SiLU(FC(GAP(x)))))
- sigma -> 1 (깨끗한 입력): 동적 경로 활성화 (최대 표현력)
- sigma -> 0 (열화된 입력): 정적 경로 활성화 (최대 강건성)

**(d) 이중 경로 보간**
- 출력 h = sigma * h_dynamic + (1 - sigma) * h_static
- 포인트와이즈 컨볼루션 및 잔차 연결(residual connection)을 거쳐 최종 출력

**(e) 상위집합 보장 (Superset Guarantee)**
- sigma = 1로 설정 시 종래의 동적 컨볼루션과 동일하게 동작
- 따라서 NC-Conv는 종래 모델 대비 성능이 저하되지 않음이 수학적으로 보장됨

#### 제2 발명: 양방향 시간축 NC-SSM (Bidirectional Temporal NC-SSM)

동영상 처리를 위한 양방향 시간축 NC-SSM은 다음 구성요소를 포함한다:

**(a) 순방향 시간축 경로 (Forward Temporal Path)**
- 시간축 고정 Conv1d (LTI 시간 평활화) 및 선택적 Conv1d (입력 의존적 시간 동역학)
- 순방향 sigma 게이트로 품질 기반 보간
- 깨끗한 과거 프레임의 문맥을 열화된 현재 프레임으로 전파

**(b) 역방향 시간축 경로 (Backward Temporal Path)**
- 입력 시퀀스를 시간축 반전하여 동일 구조 적용 후 재반전
- 깨끗한 미래 프레임의 문맥을 열화된 현재 프레임으로 전파
- 순방향만으로는 불가능한 열화->깨끗 전환 구간의 문맥 복원

**(c) 양방향 출력 결합**
- 순방향 및 역방향 출력을 연결(concatenation) 후 프로젝션
- 모든 열화 프레임이 최소 한 방향의 깨끗한 문맥을 수신

#### 제3 발명: 프레임별 품질 게이트 (Per-Frame Quality Gate)

**(a) 품질 게이트 구조**
- g_t = Sigmoid(MLP(f_t^spatial)) in [0, 1]
- f_t^out = f_t^spatial + g_t * (f_t^temporal - f_t^spatial)
- g_t -> 0: 시간축 보정 비적용 (깨끗한 프레임 보호)
- g_t -> 1: 시간축 보정 최대 적용 (열화 프레임 복원)

**(b) 게이트 초기화**
- 편향(bias)을 -3으로 초기화하여 Sigmoid(-3) ≈ 0.05
- 기본적으로 시간축 보정을 적용하지 않아 깨끗한 프레임을 보호

**(c) 비지도 품질 학습 (Implicit Quality Learning)**
- 별도의 품질 라벨 없이 분류 손실(classification loss)만으로 학습
- 클린 앵커 손실(clean anchor loss): 깨끗한 프레임의 특징이 변경되지 않도록 제약
- 게이트 정규화: 깨끗한 프레임에서 게이트 값이 0에 가깝도록 유도
- 학습된 게이트 값이 실제 열화 수준과 상관관계를 보임 (실험적 검증)

#### 제4 발명: 양방향 학습-단방향 추론 방법 (Train Bidirectional, Deploy Causal)

**(a) 학습 시**: 양방향 시간축 NC-SSM 사용 (순방향 + 역방향)
- 양방향 문맥으로 최적의 그래디언트 흐름 확보
- 순방향 경로가 양방향 학습의 이점을 내재화

**(b) 추론 시**: 순방향(causal) 경로만 사용
- 1 프레임 지연(latency)으로 스트리밍 처리 가능
- 실시간 ADAS 요구사항 충족
- 역방향 경로 제거로 연산량 50% 절감

#### 제5 발명: 2단계 학습 방법 (Two-Phase Training)

**(a) 제1단계**: 공간축 NC-Conv 사전학습
- 단일 프레임 분류 작업으로 공간축 특징 추출기 학습
- 부패 증강(corruption augmentation)과 함께 학습 (30% 비율)

**(b) 제2단계**: 시간축 NC-SSM 학습
- 공간축 모델 동결(freeze)
- 시간축 NC-SSM 및 품질 게이트만 학습
- 시간축 그래디언트가 공간축 표현을 손상시키는 것을 방지

#### 제6 발명: MCU 배포를 위한 경량화

**(a) 모델 크기**
- NC-Conv 공간축: 253K 파라미터 (INT8: 253KB)
- 시간축 NC-SSM: ~20K 파라미터 (보틀넥 구조)
- 총 ~275K 파라미터 (INT8: 275KB)

**(b) 타겟 하드웨어**
- STM32H743 (ARM Cortex-M7, 480MHz, 1MB SRAM, ~$8)
- NXP i.MX RT1170 (Cortex-M7, 1GHz, 2MB SRAM, ~$10)

**(c) 추론 성능**
- STM32H743 기준: ~35ms 지연, ~28 FPS
- 메모리 사용: 585KB / 1024KB (57%)
- 전력 소비: ~0.5W
- ADAS 실시간 요구사항(100ms 이하, 10 FPS 이상) 충족

## 발명의 효과

1. **이중 모드 적응**: 깨끗한 입력에서는 동적 경로로 최대 표현력, 열화된 입력에서는 정적 경로로 최대 강건성을 자동 달성
2. **CIFAR-10-C 벤치마크**: 동일 파라미터 표준 CNN 대비 평균 +1.6% 정확도 향상, severity 5에서 +2.9%
3. **시간축 문맥 전파**: 터널 내부 최악 프레임에서 +35.2% 정확도 회복, 깨끗한 프레임은 -1.0% 이내 보존
4. **비지도 품질 추정**: 품질 라벨 없이 분류 손실만으로 게이트가 열화 수준을 자동 학습
5. **MCU 배포**: $8 MCU에서 28 FPS 실시간 처리, 기존 $30+ CPU 대비 73% 원가 절감
6. **상위집합 보장**: sigma=1 설정 시 종래 모델과 동일, 성능 저하 없음이 수학적으로 보장
7. **모달리티 범용성**: 동일 원리가 오디오 SSM, 공간 CNN, 시간축 SSM에 적용 가능

## 청구범위

### 청구항 1 (독립항)
입력 영상을 수신하는 단계;
상기 입력 영상에 대해 고정 가중치를 사용하는 정적 컨볼루션 경로로 제1 특징맵을 생성하는 단계;
상기 입력 영상에 대해 입력 의존적 채널 게이트를 포함하는 동적 컨볼루션 경로로 제2 특징맵을 생성하는 단계;
상기 입력 영상의 품질 점수 sigma를 산출하는 단계;
상기 품질 점수에 기반하여 상기 제1 특징맵과 제2 특징맵을 보간하여 출력 특징맵을 생성하는 단계;
를 포함하는 노이즈 조건부 영상 인식 방법.

### 청구항 2
제1항에 있어서, 상기 출력 특징맵은 h = sigma * h_dynamic + (1 - sigma) * h_static으로 산출되며, sigma=1일 때 종래의 동적 컨볼루션과 동일하게 동작하는 것을 특징으로 하는 방법.

### 청구항 3
제1항에 있어서, 상기 품질 점수 sigma는 별도의 품질 라벨 없이 분류 손실(classification loss)을 통해 암묵적으로 학습되는 것을 특징으로 하는 방법.

### 청구항 4
제1항에 있어서, 동영상의 복수 프레임에 대해:
각 프레임의 공간축 특징을 추출하는 단계;
순방향 시간축 경로로 과거 프레임 문맥을 현재 프레임에 전파하는 단계;
역방향 시간축 경로로 미래 프레임 문맥을 현재 프레임에 전파하는 단계;
프레임별 품질 게이트에 기반하여 시간축 보정의 적용 정도를 제어하는 단계;
를 더 포함하는 방법.

### 청구항 5
제4항에 있어서, 상기 프레임별 품질 게이트는:
편향(bias)을 음수로 초기화하여 기본적으로 시간축 보정을 비활성화하고;
분류 손실 및 클린 앵커 손실을 통해 열화된 프레임에서만 시간축 보정이 활성화되도록 학습되는 것을 특징으로 하는 방법.

### 청구항 6
제4항에 있어서, 학습 시에는 양방향(순방향 + 역방향) 시간축 경로를 사용하고, 추론 시에는 순방향 시간축 경로만 사용하여 1 프레임 지연의 스트리밍 처리를 수행하는 것을 특징으로 하는 방법.

### 청구항 7
제1항에 있어서, 상기 방법이 1MB 이하의 SRAM을 갖는 마이크로컨트롤러(MCU)에서 INT8 양자화된 형태로 실행되며, 30ms 이하의 지연으로 실시간 추론을 수행하는 것을 특징으로 하는 방법.

### 청구항 8
제1항에 있어서, 상기 정적 경로의 출력 분산은 O(sigma_n^2)이고, 상기 동적 경로의 출력 분산은 O(sigma_n^{4+})이며, 상기 품질 점수 sigma에 의한 보간을 통해 전체 출력 분산이 O(sigma_n^2)와 O(sigma_n^{4+}) 사이에서 적응적으로 제어되는 것을 특징으로 하는 방법.

### 청구항 9 (장치 독립항)
입력 영상을 처리하는 영상 인식 장치에 있어서:
고정 가중치 컨볼루션으로 구성된 정적 경로부;
입력 의존적 채널 게이트를 포함하는 동적 경로부;
입력 품질 점수를 산출하는 시그마 게이트부;
상기 품질 점수에 기반하여 정적 경로부 및 동적 경로부의 출력을 보간하는 보간부;
를 포함하는 노이즈 조건부 영상 인식 장치.

### 청구항 10
제9항에 있어서, 동영상 프레임 시퀀스를 처리하기 위한:
순방향 시간축 상태 공간 모델부;
역방향 시간축 상태 공간 모델부;

## 요약서

본 발명은 입력 영상의 품질에 따라 컨볼루션 연산의 동작을 적응적으로 변경하는 노이즈 조건부 이중 경로 컨볼루션(NC-Conv) 블록과, 동영상에서 양방향 시간축 문맥 전파를 통해 열화 프레임의 인식 정확도를 회복하는 양방향 NC-SSM을 제안한다. NC-Conv는 깨끗한 입력에서는 입력 의존적 동적 경로(sigma->1)를, 열화된 입력에서는 고정 가중치 정적 경로(sigma->0)를 자동 선택하며, 품질 라벨 없이 분류 손실만으로 시그마 게이트가 입력 품질을 학습한다. 양방향 시간축 NC-SSM은 깨끗한 과거 및 미래 프레임의 문맥을 열화 프레임에 전파하되, 프레임별 품질 게이트로 깨끗한 프레임은 보호한다. CIFAR-10-C 벤치마크에서 동일 파라미터 CNN 대비 +1.6% 향상, 터널 시뮬레이션에서 최악 프레임 +35.2% 회복을 달성하며, $8 MCU에서 28 FPS 실시간 처리가 가능하다.
