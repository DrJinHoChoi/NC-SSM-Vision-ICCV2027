Top Improvements

Bidirectional NC-SSM

Clean-to-degraded context transfer

Frame-independent models lose accuracy on degraded frames with no recovery path. Bi-NC-SSM propagates clean-frame context through temporal dynamics — +35.2% on the darkest tunnel frame while preserving clean accuracy.

Quality Gate σt — Learned without supervision

How it works:
• Clean frames: gate ≈ 0 → skip temporal (preserve accuracy)
• Degraded frames: gate ↑ → receive clean-frame context
• No quality labels needed — learned from task loss alone
